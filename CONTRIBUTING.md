# Contributing

This repository contains my personal coursework for **Introduction to FPGA Design** and is shared as part of my academic portfolio.

## Academic Integrity

If you are currently taking this or a similar course, please do not copy any part of this work — code, reports, or images — for your own submissions. Doing so may violate your institution's academic integrity policy. Feel free to use it as a reference for understanding the concepts, but write your own implementation and report.

## Issues & Suggestions

This repo isn't actively seeking code contributions, but if you spot an error, have a suggestion, or want to discuss the implementation, feel free to open an issue or reach out via the contact info in the main [README](README.md).
